# 安全范围

## 授权范围

本仓库只实现和验证 Phoenix 框架自身的防御能力、公开 API、安全默认值与本地示例。测试目标限于仓库内代码、自动生成的临时数据和本地回环监听器。

## 允许的工作

- 协议、认证、会话、密码学、输入验证、日志脱敏和错误边界的单元/集成测试。
- 使用公开测试向量、随机测试密钥和本地临时数据库验证实现。
- 依赖审计、模糊测试、静态分析和公开 API 安全评审。

## 禁止的工作

- 对第三方服务、账户、域名、客户端或生产环境进行扫描、攻击或绕过。
- 在仓库、日志、fixture、错误消息或文档中保存真实密码、token、Cookie、私钥、客户数据或内部地址。
- 实现 keygen、破解、凭据窃取、持久化、隐蔽控制或可复用的第三方绕过工具。

## 密钥与隐私规则

- 文档和测试只使用明显的别名与随机/固定测试 key；真实 key 由部署环境注入并定期轮换。
- Authorization、Cookie、JWT、密码、明文业务 payload 与加密 key 不得进入结构化日志。
- 原始抓包、内存 dump、大型日志和真实数据默认不进入 Git。

## 停止条件

若验证需要真实第三方资产、生产凭据、不可逆数据变更或超出本仓库防御目标的攻击行为，必须停止并重新确认授权与最小测试边界。
