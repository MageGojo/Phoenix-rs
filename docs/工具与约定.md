# 工具与约定

供断点续作与 subagent 恢复上下文使用。

## Agent Skill（写应用必读）

用 Phoenix 开新项目 / 写业务时，**AI 默认必读**（已入库，见根 `README.md` / `AGENTS.md`）：

- 仓库内：`.cursor/skills/phoenix/SKILL.md`（及 `api-rust.md` / `api-react.md`）— **Git 跟踪**
- 根目录：`AGENTS.md`（强制约定入口）
- 本机副本：`~/.cursor/skills/phoenix/`（便于框架外的 `px new` 应用触发）

触发词：Phoenix、`px new` / `px make`、Laravel 风格 Rust Web、`@phoenix/react`、契约 / PageEnvelope。

发布准备见 [RELEASE.md](RELEASE.md)（整理清单；**不要**自动 `git push`）。

## 命令

```bash
cargo test --workspace --locked
cargo clippy --workspace --all-targets --locked -- -D warnings
cargo fmt --all -- --check
```

应用侧常用：

```bash
cargo install px          # crates.io 发布后；或 cargo install --path crates/phoenix-cli
px new my-app
px make:model Post --all
px migrate
px dev
```

可选：

```bash
PHOENIX_TEST_REDIS_URL=redis://127.0.0.1/0 cargo test -p phoenix-redis --locked
PHOENIX_TEST_POSTGRES_URL=postgres://... cargo test -p phoenix-database --locked
```

品牌：**Phoenix-rs**；CLI 包 `px`；门面包 `phoenixrs`（见 ADR-009）。  
托管：GitHub + GitCode 双镜像（见 `docs/RELEASE.md` / README「源码镜像」）。

## 依赖约定

| 能力 | crate / 工具 |
| --- | --- |
| HTTP | Hyper 1.x + hyper-util |
| ORM | Toasty 0.8（sqlite / postgresql / mysql） |
| Redis | `redis`（tokio + connection-manager），原子操作用 Lua |
| WebSocket | `tokio-tungstenite` + Hyper `OnUpgrade`（首版仅 H1） |
| 队列 | `phoenix-queue`（Memory backend 首版） |
| 邮件 | `phoenix-mail`（MemoryTransport 首版，无内置 SMTP） |
| 应用 CLI | `phoenix-console`（`myapp <cmd>`，`commands!` 宏） |
| 前端 | Vite + React + TypeScript |
| 配置补全 | JSON Schema + Taplo / Even Better TOML（`schemas/`、`taplo.toml`） |
| Feature / 插件 | `phoenix-plugin`：`Plugin` + `FeatureSet`（见 FEATURES.md） |
| 发版流水线 | `phoenix-release` + `px release*`（见 RELEASE_PIPELINE.md） |
| 密钥扫描 | Gitleaks + `.gitleaks.toml` |
| 依赖策略 | `deny.toml` + cargo-deny |
